CSS Cheat Sheet

by: Fernanda Cantu A01782232

border

Permite definir en una única regla todos los bordes de los elementos seleccionados. Border es una propiedad abreviada para border-color, border-style, border-width

border-color define el color
border-style define el estilo
border-width define el grosor.

border-left, border-right, border-top, border-bottom

Border-left se usa para cambiar el estilo del borde izquierdo de un elemento.

Border-right se usa para cambiar el estilo del borde derecho de un elemento.

Border-top se usa para cambiar el estilo del borde superior de un elemento.

Border-bottom se usa para cambiar el estilo del borde inferior de un elemento.

border-radius

Border-radius sirve para redondear las esquinas del borde de un elemento.

background-image

Background-image sirve para poner una imagen como fondo. hr

color

Color sirve para cambiar el color de un elemento.

font

Font sirve para cambiar y especificar la fuente o tipo de letra de un texto.

font-family

Font-family sirve para cambiar el tipo de letra de un texto de manera general.

font-style, font-weight

Font-style sirve para especificar como una fuente debe ser estilizada, ya sea normal, italica u oblicua.

Font-weight sirve para especificar que tan negrita o italica un texto debe de aparecer.

width, height

Width sirve para definir el ancho de un elemento.

Height sirve para definir la altura de un elemento.

Hay una diferencia de 100 pixeles entre este texto y el anterior.

list-style

  • todas las propiedades
  • de estilo

margin

Margin permite definir el area de los cuatro margenes de un elemento.

padding

Padding sirve para establecer un relleno alrededor del contenido de un elemento dentro de los bordes definidos para este.

overflow

Overflow sirve para definir lo que se quiere que pase con contenido que excede un elemento. Por ejemplo, este texto excede su elemento y por lo tanto se puso un overflow-scroll para agregar desplazamiento.

display

Display especifica si un elemento es tratado como block o como inline element.

position

Position sirve para especificar como un elemento es posicionado en un documento. Por ejemplo, este elemento esta en position: fixed, lo cual quiere decir que permanecerá al fondo de la página sin importar el scroll.

filter

Filter sirve para definir efectos visuales en un elemento, como saturación o enfoque.
